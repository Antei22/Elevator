# Elevator (Java)

Многопоточная симуляция системы управления лифтами в многоквартирном доме.

## Запуск



```bash
cd src
javac elevator/*.java
java elevator.Main

## Использование:

Введите количество этажей и количество лифтов в доме

Далее вводите запросы пассажиров в формате:
fromFloor toFloor

## Реализация

Каждый лифт реализован как отдельный поток

Диспетчер реализован как отдельный поток и распределяет запросы между лифтами

Обеспечен потокобезопасный доступ к состоянию лифтов

Реализована обработка некорректного пользовательского ввода

## Дополнительно реализовано

Лифт может останавливаться на этажах по пути движения

При назначении запросов учитывается занятость лифтов

Добавлено наглядное логирование движения лифтов и остановок
